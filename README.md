# 🚧 AI-Based Road Damage Detection System

A deep learning-based web application that detects road cracks from images using computer vision and MobileNetV2. The system provides real-time predictions, confidence scores, and analytics through an interactive dashboard.

---

## 🔍 Overview

Road infrastructure monitoring is critical for safety and maintenance. Manual inspection is time-consuming and error-prone.

This project automates road inspection using AI by:
- Detecting cracks in road images
- Providing confidence-based predictions
- Logging results for analysis
- Visualizing trends through dashboards

## 🧠 Features

- 📸 Upload road images
- 🤖 Crack detection using deep learning
